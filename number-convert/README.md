# 金额数字大小写转换工具

这是一个将数字金额转换为中文大写形式的工具。例如：将"199900.00元"转换为"壹拾玖万玖仟玖佰元整"。

## 功能特点

- 支持将数字金额转换为中文大写形式
- 支持到亿级别的转换
- 支持小数点后两位（角、分）
- 正确处理零和单位的显示
- 美观的用户界面

## 环境要求

- JDK 23
- Maven 3.9.9
- JavaFX（已通过Maven依赖管理）

## 构建与安装方法

### 方法1：创建可执行EXE文件（推荐）

使用以下命令创建Windows可执行文件：

```
cd number-convert
mvn clean package -P exe
```

构建完成后，您可以在`target`目录下找到`金额转换工具.exe`文件。

### 方法2：仅构建JAR文件

如果仅需要JAR文件而不需要EXE：

```
cd number-convert
mvn clean package
```

构建完成后，在`target`目录下会生成以下文件：
- `金额转换工具.jar`：基本JAR文件
- `金额转换工具-fat.jar`：包含所有依赖的胖JAR文件

## 运行方法

### 方法1：直接运行EXE文件（推荐）

双击`target`目录下的`金额转换工具.exe`文件即可运行程序。

EXE文件会自动使用您系统上的JDK 23和JavaFX模块。确保正确设置了JAVA_HOME环境变量。

### 方法2：使用java -jar命令运行JAR文件

如果您选择不创建EXE文件，也可以使用以下命令运行JAR文件：

```
java --module-path "%JAVA_HOME%\lib\modules" --add-modules javafx.controls,javafx.fxml -jar target\金额转换工具-fat.jar
```

### 方法3：使用Maven运行

也可以使用Maven JavaFX插件运行：

```
mvn javafx:run
```

## 使用说明

1. 在输入框中输入数字金额（最多支持到亿级别，小数点后最多两位）
2. 点击"转换"按钮
3. 查看转换结果，包括小写金额和大写金额

## 示例

输入：199900.00
输出：
- 小写金额：199,900.00元
- 大写金额：壹拾玖万玖仟玖佰元整

## 常见问题

1. **EXE文件无法启动**
   
   确保您的系统已安装JDK 23，并正确设置JAVA_HOME环境变量。
   如果仍然无法运行，尝试使用方法2或方法3来运行程序。

2. **使用JAR文件启动时报错"缺少JavaFX运行时组件"**
   
   这是因为JavaFX不再包含在标准JDK中。请使用完整的`java -jar`命令，包含`--module-path`和`--add-modules`选项。

3. **数字转换不正确**
   
   请确保输入格式正确，只支持数字和小数点，小数点后最多两位。

## 注意事项

- 本工具仅支持简体中文大写金额格式
- 最大支持到亿级别的数字
- 小数点后最多支持两位（角、分）
